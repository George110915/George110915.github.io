C++ 变量与数组

变量没啥好讲的，讲下类型和格式得了。

变量类型：

  • int: 整数类型；范围大约为 。
  • long long: 长整型；范围大约是 。
  • unsigned int/long long: 不包括负数，整数范围扩大一倍。
  • double: 小数类型（比 float 进度高），大约能包含 为小数。
  • char: 字符类型，能存储且仅能存储 1 个字符。
  • string: 字符串类型，能存储字符串。

数组定义：

数组就是很多个变量。

单维数组

格式：数组类型 数组名[数组大小]。

例：开一个长度为 100 的整数型数组 a; int a[100]。

注：数组的索引是从零开始的，所以你只能使用 ，所以一般 OIer 们会习惯性开大一点点。

调用数组的一个这样写即可：数组名[调用索引]。

例：调用 a 数组的第 3 个元素，a[2]。

多维数组

格式如下：数组类型 数组名[数组大小][x维数组大小]......

你可以将一个二维数组布局理解成：

调用二维数组的一个这样写即可，多维以此类推：数组名[调用索引][调用索引]。

例：调用二维数组 中第 3 行的第 2 个元素：a[2][1]。

局部和全局

局部变量仅能在定义变量的局部使用且初始化为随机数。

全局变量能在全部代码中使用用且初始化为 0。
